

## 小潘穿搭搭配页面 — 实现计划

### 整体风格
柔粉色 + 浅灰色的简约清新设计，响应式适配手机和电脑，所有交互带轻微动画。

### 页面结构（从上到下4个区域）

**1. 顶部：场景切换栏**
- 大标题「我的私人穿搭师」
- 一排圆角场景按钮：通勤日常、旅游出行、健身运动、约会穿搭、休闲居家
- 选中按钮柔粉色高亮，其他灰色；切换场景时清空搭配区

**2. 中间：全身搭配预览区**
- 横向排列5个方框：帽子、上衣、下装、首饰、场景标签
- 空状态显示虚线边框 + 对应图标
- 选中衣物后显示衣物图片填满方框
- 手机端自动改为竖向/网格布局

**3. 快捷操作栏**
- 「随机搭配」按钮（柔粉色）：从衣橱各分类随机选一件填入搭配框
- 「清空搭配」按钮（白色边框）：一键清空所有搭配框

**4. 底部：我的衣橱**
- 分类筛选标签：全部、帽子、上衣、裙子、裤子、首饰
- 网格卡片展示所有衣物（内置示例数据，含各分类衣物图片）
- 卡片悬停轻微放大，选中高亮
- 点击卡片自动根据分类填入对应搭配框，再次点击同类替换

### 数据与交互
- 使用内置 mock 数据（每个分类预设若干示例衣物，用 placeholder 图片 + emoji 区分）
- 所有状态用 React state 管理，无需后端
- 点击、切换、填充均带过渡动画

### 响应式设计
- 桌面端搭配框横向排列，衣橱4列网格
- 手机端搭配框改为2-3列网格，衣橱2列，按钮加大
